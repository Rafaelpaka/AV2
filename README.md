

Para rodar o sistema deve acessar o backend 
cd "pasta Api"
Baixar o .Net 9.0: https://dotnet.microsoft.com/pt-br/download/dotnet/thank-you/sdk-9.0.308-windows-x64-installer
e rodar o comando > dotnet build após isso se não ocorrer nenhum erro rodar dotnet run


para o front end deve se baixar o node.js : https://nodejs.org/pt/download
rodar o comando npm install e na pasta do front rodar npm run dev, acessar o link: http://localhost:5173/
cada usuário tem um carrinho diferente


Para a criação de produtos http://localhost:5259/swagger/index.html
os produtos são armazenados na pasta DatabaseJson
